# Scopal Firm Website

Astro 6 + Tailwind v4 + Vercel project for https://scopalfirm.com.

Local development:
- `npm install` — install dependencies
- `npm run dev` — start dev server at http://localhost:4321
- `npm run build` — produce production `dist/`
- `npm run preview` — serve `dist/` locally

See `.planning/` for project plans, requirements, and architecture decisions.
See `.claude/CLAUDE.md` for working norms (the user is a non-developer attorney; explain commands in plain English before running).
