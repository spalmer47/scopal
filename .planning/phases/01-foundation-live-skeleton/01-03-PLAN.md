## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Public visitor → Vercel edge CDN | Untrusted GET requests; security headers from `vercel.json` are the first response-time defense |
| Developer machine → git repo | Untrusted local files (could include `.env`); `.gitignore` is the gate |
| Build process → `dist/` | Output served to the public; must not contain secrets |
| Search-engine crawler → site | Disallowed paths must be in `robots.txt`; sitemap reachable |
